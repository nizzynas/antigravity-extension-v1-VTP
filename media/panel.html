Say What happens
hey antigravity Starts recording & appends to your prompt buffer.
send it / send the prompt Sends directly into Antigravity — no click needed.
enhance this prompt Gemini rewrites it. Approve / Reject / Try Again inline.
approve / reject / try again Voice-control the enhancement review.
clean it up / scrub that Strips filler words — never rewrites.
cancel / clear that Discards the transcript buffer.
Pause & Resume
pause / mute Mutes mic immediately. Buffer preserved.
resume / I'm back Wakes back up exactly where you left off.
